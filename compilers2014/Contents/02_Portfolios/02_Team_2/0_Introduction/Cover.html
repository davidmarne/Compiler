Java

Reason for Language Choice

We chose to use Java, since it was the programming language we all shared in common. The added benefits of using java is that it is an object oriented language. Object-oriented languages make it easier to maintain and modify existing code. For example, as we progressed through developing our compiler we were able to add new classes that we could easily reference throughout out our existing code. Overall, this made it easier to structure and build our compiler.
